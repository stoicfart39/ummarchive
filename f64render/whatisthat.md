The f64render addon in this repo has a few changes to improve texture for my purposes: 

- Trilinear filtering orientation is patched from the main repo
- Trilinear filtering more accurately blends between pixel values
- There is a slider for x and y displacement of textures, 
independent of the UV mapping. 0.09 on each is accurate to console

Done with claude in like 2 seconds. Nobody want to actually edit this shit

Only tested on my machine, you may run into issues . Install as you would the vanilla f64render addon
